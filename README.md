DISCORD AUTO CHAT

## FITUR
- Hanya 1x Chat cocok untuk push faucet pada DC
- Hanya 1 channel
- Multi akun
- Auto looping (Input manual jeda)

## CARA INSTALL
1. CLONE BOT
```
git clone https://github.com/kangrekt/discord_chat.git
cd discord_chat
```
2. INSTALL BAHAN
```
npm init -y
npm install readline-sync axios
```
3. JALANKAN BOT
```
node index.js
```
